# Latihan.1
Cara Membuat Repository
1. Login Ke github.com
2. klik simbol (+) di pojok kanan atas
3. klik new repository
4. Silahkan beri nama projek dengan tidak memakai spasi. Misalnya "Latihan.1"
5. Setelah itu akan muncul kode komando untuk membuat repository local.
6. Ke Desktop, setelah itu klik kanan dan klik Git Bash Here
7. Setelah itu ketik "cd D:" (Tanpa tanda kutip)
8. ketik "mkdir Latihan.1" Kemudian ketik lagi "cd Latihan.1"
9. Ketika sesuai dengan apa yang muncul pada nomor 4
10. Setelah selesai, refresh pada browser anda.
11. Jika berhasil maka akan muncul README.md
12. Selesai.
